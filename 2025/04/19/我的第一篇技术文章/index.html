开启我的博客之旅

发表于 2025-04-19 分类于 建站记录

欢迎来到我的新博客！

经过一番折腾，我终于使用 Hexo 搭建了这个基于 GitHub Pages 的静态博客。

为什么选择 Hexo？

  • 快速、简洁且高效： 生成速度快，部署简单。
  • Markdown 支持： 可以专注于内容创作。
  • 丰富的主题和插件： 强大的社区支持。

接下来，我计划在这里记录一些：

  1. 技术学习笔记
  2. 项目实践心得
  3. 生活随想感悟

希望这个小站能伴随我一起成长！
